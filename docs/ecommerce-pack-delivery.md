# E-commerce Pack Public Demo Boundary

This document describes the public demo boundary for the ecommerce support pack. It is not a customer delivery playbook.

## Public Purpose

The ecommerce support pack demonstrates how Yutra models a support-oriented agent with:

- mock/demo inputs
- mock adapters
- policy examples
- response templates
- trace/audit/certification evidence

## Public Validation

Developers can use the pack to:

- inspect the example DSL
- run local mock scenarios
- export trace/audit evidence
- review adapter contracts
- understand how public demo assets connect to certification

## Not Included

- customer-specific SOP
- pricing or proposal material
- production adapter mapping
- UAT or rollout playbooks
- private delivery scope
- real customer data, credentials, or endpoints

Commercial implementation assets should live outside this public repository.
