# E-commerce Public Demo Risk Boundary

This public document lists generic demo limitations. It is not a customer delivery risk register.

## Demo Limitations

- Mock adapters do not represent production reliability.
- Demo policies are not customer-approved policy.
- Demo templates are not brand-approved customer messaging.
- Local trace/audit output is for developer validation only.
- Example cases are not exhaustive production coverage.

## Public Safety Boundary

The public repository intentionally excludes:

- real customer endpoints
- credentials or API keys
- production adapter mapping
- customer-specific SOP
- production UAT or rollout risks
- commercial delivery commitments

Use private implementation materials for customer-specific risk management.
