# E-commerce Support Pack Scope Checklist (P3-04)

## A) Included in this delivery scope

- shipping / return / refund / handoff SOP paths
- policy parameter layer and environment profile samples
- knowledge templates and response templates
- adapter contracts and mock adapters
- trace, audit bundle, and certification linkage

## B) Explicitly out of scope

- customer-service backend or seat-management system
- BI/operations dashboard
- multi-store/multi-tenant support platform
- full CRM/OMS/ERP integration
- pre-sale recommendation and marketing automation

## C) Must confirm before delivery

- customer channel priorities
- API readiness for order/shipping/return/refund/ticket
- policy parameter baseline
- template wording and legal review needs
- handoff ownership and escalation queue
- acceptance criteria for shipping/return/refund/handoff
