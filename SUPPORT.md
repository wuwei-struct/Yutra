# Support

## Supported Scope

- Local development with pnpm workspace.
- DSL validation and runtime execution in repository examples.
- Local JSONL trace viewing with `@yutra/viewer`.
- Skill-based demo path and ecommerce skill pack usage.

## Not Supported

- Platformization requirements (tenant/account/permissions/ops console).
- Hosted API/server support.
- Cloud LLM integration support in this reference stage.
- Marketplace / remote registry / install workflow requests.

## Getting Help

- Usage and contribution flow: `README.md` and `CONTRIBUTING.md`.
- Skill-based architecture docs: `docs/skill-based-runtime.md`.
- Demo walkthrough: `docs/skill-based-demo-path.md`.
- Security guidance: `SECURITY.md`.
